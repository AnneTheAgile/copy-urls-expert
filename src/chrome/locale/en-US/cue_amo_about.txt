<strong>Overview</strong>

Copy Urls Expert comes in handy when you want to copy a bunch of urls. It provides you a few possibilities:
<ol>
 <li> Copy urls of all tabs in every opened window.
 <li> Copy urls of all tabs only in the current window.
 <li> Copy urls of all tabs in the current tab group.
 <li> Copy urls of all links in the selection.
 <li> Copy urls of all images in the selection.
</ol>

You can also open multiple links at once by using 'Open Tabs' menu.

To access these functions, find 'Copy Urls Expert' menu at following places:
<ul>
<li> Menubar -&gt; Edit Menu
<li> Firefox AppMenu -&gt; Edit Menu
<li> Tab Context Menu
<li> Selection Context Menu
<li> Copy Urls Expert Toolbar button (You will have to add button manually. <a href="https://support.mozilla.org/en-US/kb/customize-firefox-controls-buttons-and-toolbars#w_how-do-i-customize-or-rearrange-toolbar-items">How to add button</a>?)
</ul>
<strong>Customization</strong>

The format in which links are copied to clipboard is highly customizable. See the extension's options dialog. Following parameters allow you to customize each url:

$url - Actual link e.g. http://google.com/
$title - Title of the tab e.g. Google
$n - New Line.
$t - Tab character.
$index - Serial number of link.
$date - Today's date
$date(format_string) - Today's date/time is the format specified by format string:
             YYYY - four digit year
             mm - two digit month
             dd - day of the month
             HH - hour in 24-hour format
             hh - two digit hour
             h  - hour
             MM - two digit minute
             M  - minute
             SS - two digit second
             S  - second

Remember that by default urls are not placed on new lines so $n must be used anyways.

<strong>Bugs, Support</strong>

Please report bugs at the <a href="https://github.com/kashiif/copy-urls-expert/issues">GitHub Issue Tracker</a>.
